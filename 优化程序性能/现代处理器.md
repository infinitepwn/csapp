延迟latency表示运算完成所需要的总时间
发射时间issue time表示,连续的同类型运算之间需要的最小时钟周期数
容量capacity,能执行该运算的功能单元的数量

最大吞吐量定义为发射时间的倒数
**Pipelined Functional Units**（流水线功能单元)
``` C
long mult_eg(long a, long b, long c) {
    long p1 = a*b;
    long p2 = a*c;
    long p3 = p1 * p2;
    return p3;
}
```
